# Whack-a-Mole Game

このゲームはGitHub Pagesで公開されています。以下のリンクからブラウザで直接遊ぶことができます。
**[ここをクリックしてゲームをプレイ](https://soichiro1357533.github.io/Whack-a-mole/)**


これはシンプルなモグラ叩きゲームです。

## 遊び方

1.  リンクを開くか、ローカルで `index.html` ファイルをウェブブラウザで開きます。
2.  ホーム画面でモードを選択してゲームを開始します。
3.  ランダムな穴から飛び出す先生をクリックして叩きます。
4.  先生を叩くとスコアが増え、一定スコアに達するとレベルアップしてスピードが上がります。
5.  時々出現するアイテムを叩くと、有利な効果が得られます。

## ゲームモード

このゲームには2つのモードがあります。

### 1. ノーマルモード (Normal Mode)
*   **ルール**: 30秒の制限時間内にどれだけスコアを稼げるか挑戦します。
*   **終了条件**: 制限時間（30秒）が経過するとゲーム終了です。
*   **アイテム**: ⏰ (時計) - 残り時間が5秒増えます。

### 2. エンドレスモード (Endless Mode)
*   **ルール**: ライフが尽きるまで続きます。初期ライフは3つ（❤）です。
*   **ライフ減少**: 先生が穴に戻るまでに叩けなかった場合、ライフが1つ減ります。
*   **終了条件**: ライフが0になるとゲーム終了です。
*   **アイテム**: ❤ (ハート) - ライフが1つ回復します。
